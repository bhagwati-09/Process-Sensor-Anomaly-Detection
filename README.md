# Process-Sensor-Anomaly-Detection

Implemented unsupervised process fault detection on the Tennessee Eastman Process benchmark using Isolation Forest and a custom Autoencoder neural network. 
Achieved 95% F1-score on Fault 1 detection with reconstruction error showing a 16x gap between normal and faulty operating conditions. 
Built end-to-end: data preprocessing, model training, threshold selection, and performance visualization across 52 process sensors.
